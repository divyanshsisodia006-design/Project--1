# First project - Saini Shudh Bhojanalya

## 🥗 Project Description

**Saini Shudh Bhojanalya** is a vegetarian restaurant website dedicated to serving **pure veg packed foods only**.  
The project highlights a clean, user‑friendly design where customers can explore the menu, learn about the restaurant’s values, and place orders for hygienically packed meals.  

### Key Highlights
- Focus on **pure vegetarian cuisine**  
- **Packing‑only service** ensuring freshness and convenience  
- Simple navigation for browsing menu items  
- Responsive design for mobile and desktop users  
- Easy integration for online ordering and delivery partners  

This website aims to reflect the restaurant’s commitment to **quality, hygiene, and authentic taste**, while offering a modern digital presence for customers.
